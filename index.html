关于我们


OpenAI ChatGPT API 成本计算器（中文版）是一款专门为团队和开发者设计的工具，可以帮助用户准确预算和规划他们在项目中使用ChatGPT API的成本。

主要特点和功能：


1) 基于中文汉字的 API 价格/成本计算器，对中文开发者更加友好；
2) 可以帮助用户估算项目成本，规划API使用；
3) 精确性高，简单易懂。

计算细节：


1) 模型对应的价格：

GPT-3.5-Turbo 4K - $0.002 / 1K tokens
GPT-3.5-Turbo 16K - $0.004 / 1K tokens
GPT-4.0 8K - $0.06 / 1K tokens
GPT-4.0 32K - $0.12 / 1K tokens

2) 计算公式：

成本 = (中文字数 * 2 / 1000) * 模型价格
